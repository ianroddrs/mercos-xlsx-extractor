# **Extrator de SKUs e Quantidades - MERCOS**

## **Sobre o Projeto**

Este projeto consiste em uma extensão de navegador desenvolvida sob demanda exclusiva para a **MLR REPRESENTAÇÕES LTDA**.

O principal objetivo da ferramenta é promover a automação no processo de extração de dados de pedidos. A extensão identifica, coleta e estrutura automaticamente os códigos de identificação (SKU) e as respectivas quantidades dos itens, otimizando de forma significativa o fluxo de trabalho necessário para o preenchimento de planilhas de controle e gestão de pedidos.

## **Guia Rápido de Instalação**

A instalação varia de acordo com o dispositivo e o navegador utilizado. Siga os passos correspondentes ao seu caso de uso.

### **Instalação em Computadores (Google Chrome ou Microsoft Edge)**

Esta instalação utiliza o código-fonte descompactado, ideal para ambientes desktop.

- Faça o download do **Código-fonte do Projeto (.zip)** no link acima.
- Extraia (descompacte) o conteúdo do arquivo ZIP em uma pasta de fácil acesso em seu computador.
- Abra o seu navegador (Google Chrome ou Microsoft Edge).
- Na barra de endereços, digite o seguinte comando e pressione Enter:
  - No Chrome: chrome://extensions/
  - No Edge: edge://extensions/
- No canto superior direito da tela de extensões, ative a opção **"Modo do desenvolvedor"** (ou "Developer mode").
- Novos botões aparecerão na parte superior. Clique no botão **"Carregar sem compactação"** (ou "Load unpacked").
- Navegue pelas pastas do seu computador e selecione a pasta onde você extraiu os arquivos do passo 2.
- A extensão será carregada imediatamente e o ícone aparecerá na barra de ferramentas do navegador, pronta para uso.

### **Instalação em Dispositivos Móveis (via Microsoft Edge Canary)**

Esta instalação utiliza o arquivo compilado .crx e requer uma versão específica do navegador Edge para Android.

- Acesse a loja de aplicativos do seu dispositivo móvel (Google Play Store) e instale o navegador **Microsoft Edge Canary**.
- Faça o download do **Arquivo de Instalação (.crx)** fornecido na seção de Downloads diretamente no seu dispositivo móvel.
- Abra o aplicativo Microsoft Edge Canary.
- No menu de configurações selecione "Sobre o Microsoft Edge" e clique na versão do aplicativo até ativar as opções de desenvolvedor.
- Nas opções de desenvolvedor selecione a opção "Extension install by crx" e selecione o arquivo .crx que foi baixado em seu dispositivo.
- A extensão estará instalada e acessível através do menu do navegador Edge Canary.
